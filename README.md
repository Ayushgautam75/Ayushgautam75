# 💫 About Me 👨‍💻

I’m Ayush, a Data Science and Generative AI enthusiast with a strong foundation in Python, Machine Learning, Deep Learning, NLP, and LLMs. I enjoy building end-to-end AI solutions — from data preprocessing and model training to deployment.

I have hands-on experience with data analysis, visualization, and ML model deployment using tools like Pandas, NumPy, TensorFlow, PyTorch, and Flask. I’ve worked on projects involving chatbots, recommendation systems, NLP applications, and computer vision.

I’m continuously learning and aiming to grow as a Data Scientist / Machine Learning Engineer / Generative AI Engineer, building scalable and impactful AI systems.
